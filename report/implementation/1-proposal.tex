\subsection{Overview of the Project}

The project will provide extra information \fxnote{statistics instead of information?} related to users and media on Kitsu, which would require dozens of expensive requests to generate client-side.

\subsection{External Web Services to Be Consumed}

Kitsu, an anime \& manga discovery and tracking service (similar to IMDb): \url{https://kitsu.io}. It exposes a RESTful API (\url{https://kitsu.docs.apiary.io}) using the JSON:API\footnote{\url{http://jsonapi.org}} specification.

      \item Total media in category
    \end{enumerate}
  \item Media

      \item Rank in category (position and percentile)

\end{enumerate}
